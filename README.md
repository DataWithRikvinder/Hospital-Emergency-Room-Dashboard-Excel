🏥 Hospital Emergency Room Dashboard (Excel)
📌 Project Overview
This project presents an interactive Hospital Emergency Room Dashboard developed in Microsoft Excel. The dashboard enables healthcare stakeholders to monitor patient flow, waiting time, department referrals, patient satisfaction, and admission trends through interactive visualizations.
🎯 Objective
The primary objective of this project is to transform raw hospital data into meaningful insights that support operational decision-making and improve emergency room performance.
